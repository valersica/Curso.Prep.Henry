*Variables: son palabras clave que contienen informacion (pueden ser palabras, numeros o acciones), que vamos  usar despues para invocar esa data.
*Strings: son cadenas o bloques de texto, se escriben entre comillas siempre. Mayormente letras, aunque pueden contener numeros (pero sirven tambien para nombrar y no para hacer calculos). Pueden ser objetos en una funcion o definir una variable.
*Funciones: son tambien palabras clave que tienen escritas dentro acciones. El argumento es el codigo que tiene escrito dentro para ejecutar esa accion. El objeto o parametro de la funcion se escribe al lado de su nombre, va entre parentesis y si es mas de uno se separa por comas. Return se usa para devolvernos los datos de justamente, la funcion que le dijimos que realice.
*Declaraciones: son los argumentos de las funciones, la descripcion de como tiene que funcionar. Declaraciones condicionales: IF se usa para que nos devuelva datos SI ciertas condiciones que escribimos estan dadas. Tambien puede ser ELSE si son mas opciones (declaraciones anidadas), por ejemplo "SI tal cosa pasa, hace esto. SINO hace tal otra"
*Booleanos: son un tipo de dato binario (tienen 2 opciones) que en JS se usa para saber si cierto dato es real (true) o falso (false).
